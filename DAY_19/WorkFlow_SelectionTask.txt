Scenario 1:

Client adjuradication

Graph flow

This process has:

Parallel execution of three independent checks:
Fraud Screening
Policy Coverage Check
Medical Coding Review
A synchronization/join step where all results are collected.
A final decision node that uses the outputs from all three checks to approve or deny the claim.

Visual flow
           Claim
              |
      -----------------
      |       |       |
   Fraud   Coverage  Medical
   Check    Check     Review
      |       |       |
      --------|--------
              |
      Decision Agent
              |
        Approve/Deny
